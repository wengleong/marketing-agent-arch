Daily Flow
7am — n8n cron
Cron fires Claude API. Reads Campaign State. Pulls GA4 + Ads. Writes Notion briefing. Pings marketer in Teams.
Morning — Brief
Marketer talks to Sidekick in Cowork. Sidekick picks agents, chains write → review → draft.
9am — n8n cron
Brief processor check. If new brief dropped overnight, Sidekick triggers write chain automatically.
Midday — Approve
Draft in Notion. Teams ping. Marketer toggles status → Approved. n8n webhook fires publish.
5pm — n8n cron
Daily digest: what went out, what's pending, one recommendation. Campaign State updated.
